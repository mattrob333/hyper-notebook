To replicate this dashboard (which appears to be Google's NotebookLM interface) using the **Claude/Anthropic Dark Mode** aesthetic, I have broken down the design into its core components and created a complete "Builder Prompt" you can give to an AI coding agent.

I have also provided the **direct code** below if you want to run it immediately.

### 1\. The Design System: "Claude Dark"

The user specifically requested "Claude dark colors" to replace the standard slate blue. Based on Anthropic's brand guidelines, here is the palette we will use:

  * **Background:** `#191918` (Warm charcoal, not cold black)
  * **Surface/Cards:** `#222221` (Slightly lighter warm gray)
  * **Text (Primary):** `#EDEDE9` (Warm off-white/beige)
  * **Text (Secondary):** `#9F9E9B` (Muted warm gray)
  * **Accent:** `#D97757` (The signature Anthropic "Clay/Terracotta" orange)
  * **Borders:** `#353533` (Subtle warm borders)

-----

### 2\. Prompt for a Builder Agent

If you want another AI (like Cursor, v0, or Bolt) to build this for you, copy and paste the block below. It contains the exact structural requirements.

> **System Instruction:**
> Build a React dashboard matching the Google NotebookLM 3-column layout, but strictly using the "Claude/Anthropic Dark Mode" color palette (Warm charcoal `#191918` background, `#D97757` clay accent, `#EDEDE9` text).
>
> **Layout Requirements:**
>
> 1.  **Header:** Full width. Title: "Agent-to-User Interface and AI Research Tools". Icons on right: Create notebook, Analytics, Share, Settings, Profile.
> 2.  **3-Column Grid:** Fixed height (viewport).
>       * **Left Col (Sources, width \~20%):** Header "Sources". Action button "Add sources". Search bars for "Deep Research" and "Web". List of file items (PDF/Link icons) with checkboxes.
>       * **Center Col (Chat/Reading, width \~55%):** Header "Chat". Large scrollable reading area with markdown text, citations in blue pills `[1]`, and bold headers. Bottom fixed input area: "Start typing..." with attachment clips. Above input: Suggested query chips (e.g., "What is A2UI's current version?").
>       * **Right Col (Studio, width \~25%):** Header "Studio". A 2x4 grid of action cards (Audio Overview, Video Overview, Mind Map, Reports, Flashcards, Quiz, Infographic, Slide Deck). Below that: List of "Saved Notebooks" (e.g., "AI-Executable PRD..."). Bottom button: "Add note".
>
> **Styling Details:**
>
>   * Use `lucide-react` for icons.
>   * Font: Inter or similar clean sans-serif.
>   * Radius: `rounded-lg` for cards, `rounded-full` for chips/buttons.
>   * **Strict Color Enforcement:** No blue/slate backgrounds. Use `#191918` for app bg, `#222221` for panels/cards, `#353533` for borders.
